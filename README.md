# Note Flow

Note Flow is a modern note-taking web application built with **Next.js**, **React**, and **Tailwind CSS**.  
It allows users to create, edit, and delete notes with a clean and responsive user interface.

<img width="1919" height="929" alt="image" src="https://github.com/user-attachments/assets/72fd5ba6-b30c-4c58-905b-3bcda4e22657" />


## Features
- User authentication (Sign up / Sign in)
- Create new notes
- Edit existing notes
- Delete notes
- Real-time UI updates
- Responsive design for all devices
- Toast notifications for user actions
- Clean and modern UI design

## Technologies Used
- Next.js
- React
- TypeScript
- Tailwind CSS
- React Query
- REST API Integration
